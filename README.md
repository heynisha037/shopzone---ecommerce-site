# ShopZone — Amazon-style Demo Store

A full-featured online shopping demo built with **Next.js**, **TypeScript**, and **Tailwind CSS**. Inspired by Amazon’s layout and UX patterns.

## Features

- **Homepage** — Hero banner, category grid, deals, best sellers, product recommendations
- **Product pages** — Images, ratings, pricing, add to cart, related items
- **Search** — Full-text search across titles and descriptions
- **Categories** — Electronics, Books, Home, Fashion, Sports, Beauty
- **Shopping cart** — Persistent cart (localStorage), quantity updates, order summary
- **Checkout** — Shipping & payment form (demo only, no real payments)
- **Deals page** — Discounted products

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Tech stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- Client-side cart with React Context + localStorage

This is a demonstration project. Product data is static and checkout does not process real payments.
